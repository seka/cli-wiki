# cli-wiki

## Description

検索ワードにヒットする文字列をwikipediaから取得し

概要を表示するターミナルコマンド.

## Usage

```
cli-wiki abst keyword
```

## Install

To install, use `go get`:

```bash
$ go get -d github.com/seka/cli-wiki
```
